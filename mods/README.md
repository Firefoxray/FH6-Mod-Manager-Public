# Mods Folder

Put installable FH6 mod payloads here.

Each mod should be a folder that mirrors the Forza Horizon 6 install directory.

Example:

    mods/my_mod/
    ├── README.txt
    └── media/
        └── UI.zip

Or for root-level mods:

    mods/my_radio_mod/
    ├── version.dll
    ├── fh6-radio/
    └── media/

This public repository intentionally does not include third-party mod files, game files, extracted `.swatchbin` files, DLLs, FMOD banks, or copied `ControllerIcons.zip` / `WheelIcons.zip` archives.

Download or build those files yourself, then place them here locally.
